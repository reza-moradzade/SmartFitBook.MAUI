# 💪 SmartFitBook.MAUI

Workout management app with **.NET MAUI Blazor Hybrid**

---

## 📱 Features

- 👥 Trainer & Trainee management
- 🏋️ 1400+ exercises with animations
- 📅 Create weekly/monthly workout plans
- 📤 Export plans as JSON files
- 📥 Import plans from JSON
- 📱 Fully offline

---

## 🛠️ Tech Stack

- .NET MAUI 8.0
- Blazor Hybrid
- Entity Framework Core + SQLite
